# Bilibili-Mac
 Bilibili 官方桌面版 macOS 移植

### 使用方法
1. 解压并将 `Bilibili.app` 移动至 `Applications` 文件夹
2. 在终端中运行 `xattr -cr /Applications/Bilibili.app`
3. 在 `系统偏好设置` -> `安全性与隐私` -> `防火墙` 中允许 `Bilibili.app` 的传入连接
